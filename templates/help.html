Help - REST API Documentation

Endpoints

  • POST /api/create_server
    Create a new Modbus server.
    Request Body: { "server_id": "string", "address": "string", "port": "integer" }
    Response: { "status": "success", "server_id": "string" }
  • POST /api/stop_server/<server_id>
    Stop a Modbus server.
    Response: { "status": "success", "server_id": "string" }
  • GET /api/get_registers/<server_id>
    Get register values for a Modbus server.
    Query Parameters: parameters=[{ "name": "string", "address": "integer", "function_code": "integer" }]
    Response: { "register_name": "value", ... }
  • POST /api/set_register/<server_id>
    Set a single register value for a Modbus server.
    Request Body: { "name": "string", "address": "integer", "function_code": "integer", "value": "integer" }
    Response: { "status": "success" }
  • POST /api/set_action/<server_id>
    Set an action for a parameter.
    Request Body: { "param_name": "string", "action": "string" }
    Response: { "status": "success" }
